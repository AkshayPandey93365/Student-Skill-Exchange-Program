# Student Skill Exchange Platform

## Overview

The **Student Skill Exchange Platform** is a web-based platform that enables students to learn new skills by exchanging knowledge with their peers. Users can showcase their expertise, discover students with complementary skills, send exchange requests, participate in community discussions, build portfolios, and receive reviews from other learners.

This project is developed as a frontend prototype using **HTML, CSS, and JavaScript**.

---

## Features

### 1. Skill Discovery

* Browse student profiles
* View skills offered and skills requested
* Find suitable learning partners

### 2. Skill Exchange Requests

* Send skill exchange requests
* Connect with students having complementary skills

### 3. Analytics Dashboard

* Track skills learned
* Monitor skills shared
* View completed exchanges
* Check overall ratings

### 4. Community Forum

* Discuss learning resources
* Ask questions
* Share knowledge with other students

### 5. Portfolio Showcase

* Display completed projects
* Highlight achievements and skills

### 6. Reviews & Endorsements

* Receive feedback from peers
* Build credibility within the platform

### 7. User Settings

* Update profile information
* Manage account details
